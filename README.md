# 📖 HariPath Website

A simple, responsive, and user-friendly **HariPath Website** built using **HTML, CSS, and JavaScript**. This website allows users to read HariPath online with useful features like Dark Mode, Search, Bookmark, Share, Print, Font Size Control, and Reading Progress.

---

## 🌟 Features

- 📖 Read HariPath Online
- 🔍 Search Text
- 🌙 Dark Mode
- 🔖 Bookmark Favorite Sections
- 📤 Share HariPath
- 🖨️ Print HariPath
- 🔠 Increase/Decrease Font Size
- 📈 Reading Progress Bar
- 📜 Smooth Scrolling
- 📱 Fully Responsive Design

---

## 🛠️ Technologies Used

- HTML5
- CSS3
- JavaScript (ES6)
